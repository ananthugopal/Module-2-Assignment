# Module-2-Assignment
This project provides a analysis of an employee dataset, focusing on  team structure, positional roles, and salary expenditure across the organization.

A series of analytical tasks were executed to explore demographic, organizational, and financial characteristics of the workforce:

Team Distribution
  Counted employees across different teams using value_counts().
  Computed percentage split relative to total staff.

Age Group Distribution
  Categorized employees into age brackets.
  Identified the predominant age group: 26–35.

Position-Level Analysis
  Mapped the number of employees in each position (SG, PF, PG, SF, C).
  Determined the position with the highest salary expenditure:
  C (Centers) → $466,377,332.

Team-Level Salary Analysis
  Summed total salary expenditure per team.
  Identified the team with the highest salary expenditure:
  Cleveland Cavaliers.

KEY INSIGHTS
  Position SG are the most common role.
  Position C are the most expensive role despite smaller numbers
  Cleveland Cavaliers have the highest salary expenditure among teams.
  The organization has a predominantly mid-career workforce (26–35).
